1.安装git ：https://git-scm.com/ 一切默认即可

![]images/11.jpg

2.访问GitHub主页https://github.com/ 注册账户

3.Username填写用户名，只允许字母数字； Email address填写邮箱，会有验证邮件发过来；填写密码后点击create an account；之后到邮箱验证。

4.验证后登录github，创建Github Pages项目，点击New repository

5.创建仓库，仓库名格式为 用户名.github.io ，然后点击 Create repository

至此github部分注册完毕。

1.下载hexo便携版 https://portablehexo.bitmoe.cn/#download 解压到我们创建的一个文件夹里。（这里需要我们自己创建一个文件夹方便快捷访问）
2.解压完成后如图

3.双击“配置GitHub部署”按照提示输入相关信息（你的GitHub用户名和邮箱 按Enter键三次后输入github的登录密码，之后命令行自动关闭）

4.双击“配置基本信息”，输入自定义的主、副标题、描述、昵称和你在GitHub上创建的项目名称（用户名.github.io) 。

5.双击“渲染并本地测试”，浏览器输入localhost:4000测试hexo效果·

7.双击“渲染并部署”，浏览器访问 用户名.github.io ，若与本地测试界面一样则成功部署到github上。

至于其他的功能写文章，换主题，调格式，增加评论赞赏功能等等，大家就学会自己动手查资料查官方文档，网上的教程一大片，这里就不详述了。
